Description

This is an annotation tool for 3d motion data generation. The major function is to annotate the location of camera and its focal points. In each scene, there is no more than one camera.

The component to mark the camera is "nplab_camera" as shown below

The component to mark the focal point is "nplab_target" as shown below

Tools

Clear all instances of compenent "nplab_camera" and all instances of "nplab_target".
Double click any place in the scene. It will place an instance of compenent "nplab_camera" there.
Double click any face in the scene. It will add an instance of component "nplab_target" glue to the face.

Save the annotation to a file

Workflow

A normal workflow to use this tool is as following:

  1. use tool to clear the model
  2. use tool to set up the camera
  3. use tool to set focal points
  4. use tool to save setting
  5. repeat this process to set up another camera for the scene
